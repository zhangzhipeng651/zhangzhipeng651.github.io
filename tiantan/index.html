天探——天体探索者

天探是吴江中学原高一（1）班天文小组开发的一款App，用来向大家科普天文知识。

特色功能

  • 天体介绍：对太阳系进行描述；
  • 答题：随机抽取问题并让用户回答，用户提交答案后，将根据答题情况评等第，让用户了解自己对天体知识的掌握情况。

特点

（此部分的语言可能有点...）

  • 使用 Java 语言编写；
  • 启动速度快：在大部分的手机上启动时间不到1秒；
  • 体积小：最新的0.1.3版本安装包体积仅有 8.28 MB 大小。
    （其实并不小，因为安装包内放了二十多张图片。）
  • 开源：本软件根据GPLv3协议开放源代码，访问GitHub仓库获取源代码。

缺点

  • 功能少：受限于作者的精力和水平，本软件目前仅有两个功能；
  • 资料匮乏；
  • 仅支持安卓平台。
